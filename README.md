# ☂️ Umbrella — Intelligent Weather Dashboard

An immersive, real-time weather intelligence dashboard built with HTML, Tailwind CSS, and JavaScript.

Umbrella transforms traditional weather forecasting into an interactive atmospheric experience through dynamic visual simulations, adaptive UI theming, procedural particle systems, and location-based live weather data.

## Features

### 🌍 Real-Time Weather Data
- Fetches live weather conditions using the OpenWeather API
- Supports any user-entered location worldwide
- Displays:
  - Temperature
  - Wind speed
  - Cloud / precipitation coverage
  - Current weather conditions

### 🎨 Dynamic Weather-Themed UI
Automatically adapts interface styling based on weather conditions:

- **Rain**
- **Clear / Sunny**
- **Snow**
- **Storm / Hail**

Each weather state updates:
- Background theme
- Accent colours
- Glassmorphism glow effects
- Atmospheric animations


### 🌧 Procedural Particle Engine
Custom canvas-rendered particle systems simulate:

- Rainfall
- Snowfall
- Hailstorms

Built using JavaScript animation loops and canvas rendering.



### ☕ Interactive Smart Features

#### Coffee Run Analyzer
Evaluates whether conditions are suitable for a tactical coffee run.

#### Pub Garden Index
Scores outdoor social viability based on weather severity.

#### Dynamic Wardrobe Architect
Generates weather-appropriate clothing recommendations.

#### Windows of Opportunity
Identifies optimal dry weather windows.
